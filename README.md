# Livre d'or

## Présentation

Voici le dépôt d'un projet `Express`, `Sequelize`, `PostgreSQL` (pour les technos principales) permettant de mémoriser des messages dans une base de données.

## Installation

1. `npm install`
2. Créer une BDD
3. renseigner le `.env`
4. lancer le script `npm run db:create`
5. lancer le server de dev avec `npm run dev`
